## Local Hot Reload Setup

You can now run this project locally with hot reload (like StackBlitz) using [Vite](https://vitejs.dev/):

1. **Install dependencies**  
   In your project directory, run:

   ```bash
   npm install
   ```

2. **Start the dev server**

   ```bash
   npm run dev
   ```

3. **Open your browser**  
   Visit the local URL shown in your terminal (usually [http://localhost:5173](http://localhost:5173)).  
   Any changes you make to the code will automatically reload in the browser.
